## Prerequisites

## Installation

Install dependencies

Copy and replace .env.example

## Usage

Run server

- python app.py

## Docker

Build app image

- docker build -t finnerio-music-api .

Run image using compose

- docker run -e ... --net=backend -p 8000:8000 finnerio-music-api
